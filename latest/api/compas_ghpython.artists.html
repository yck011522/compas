artists

Artists for visualising (painting) COMPAS objects with GHPython. Artists convert COMPAS objects to Rhino geometry and data.

pass

Geometry Artists

CircleArtist

Artist for drawing circles.

FrameArtist

Artist for drawing frames.

LineArtist

Artist for drawing lines.

PointArtist

Artist for drawing points.

PolylineArtist

Artist for drawing polylines.

Datastructure Artists

MeshArtist

A mesh artist defines functionality for visualising COMPAS meshes in GhPython.

NetworkArtist

A network artist defines functionality for visualising COMPAS networks in GhPython.

VolMeshArtist

A volmesh artist defines functionality for visualising COMPAS volmeshes in GhPython.

Robot Artist

RobotModelArtist

Visualizer for robots inside a Grasshopper environment.

Base Classes

BaseArtist

Abstract base class for all GH artists.

PrimitiveArtist

Base class for artists for geometry primitives.

ShapeArtist

Base class for artists for geometric shapes.
